# 프로젝트명

## 👀프로젝트명 & 소개 

- **프로젝트명** : **미정**
- 프로젝트 소개
    - MMORPG
    
- 개발기간 : 2022.08.31 ~ 2022.09.21
- 개발인원 : 개발자 1명
- 주요 기능
    - 아이템 & 인벤토리 시스템 : 아이템을 효과적으로 관리하기 위한 구조 설계
    - 스텟 매니저 : 스텟 관리 & 아이템에 적용
    - Ai 몬스터 매니저, 보스 몬스터 : 몬스터 pool로 관리 및 몬스터 아이템 드랍 관리
   
- 사용엔진 : Unreal Engine 4
